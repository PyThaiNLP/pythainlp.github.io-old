pythainlp.transliterate¶

The pythainlp.transliterate turns Thai text into a romanized one (put simply, spelled with English).

Modules¶

pythainlp.transliterate.romanize(text, engine='royin')[source]¶
Parameters:
  • text (str) – Thai text to be romanized
  • engine (str) – ‘royin’ (default) or ‘thai2rom’. ‘royin’ uses Thai Royal Institute standard. ‘thai2rom’ is deep learning Thai romanization (require keras).
Returns:

English (more or less) text that spells out how the Thai text should read.
pythainlp.transliterate.transliterate(text, engine='ipa')[source]¶
Parameters:
  • text (str) – Thai text to be transliterated
  • engine (str) – ‘ipa’ (default) or ‘pyicu’.
Returns:

A string of Internaitonal Phonetic Alphabets indicating how the text should read.
