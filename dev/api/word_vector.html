pythainlp.word_vector¶

The word_vector contains functions that makes use of a pre-trained vector public data.

Dependencies¶

Installation of numpy and gensim is required.

Modules¶

pythainlp.word_vector.get_model()[source]¶

Download model :return: gensim model

pythainlp.word_vector.most_similar_cosmul(positive: list, negative: list)[source]¶

Word arithmetic operations If a word is not in the vocabulary, KeyError will be raised. :param list positive: a list of words to add :param list negative: a list of words to substract :return: the cosine similarity between the two word vectors

pythainlp.word_vector.doesnt_match(listdata)[source]¶

Pick one word that doesn’t match other words in the list If a word is not in the vocabulary, KeyError will be raised. :param list listdata: a list of words :return: word that doesn’t match

pythainlp.word_vector.similarity(word1, word2)[source]¶

Get cosine similarity between two words. If a word is not in the vocabulary, KeyError will be raised. :param str word1: first word :param str word2: second word :return: the cosine similarity between the two word vectors

pythainlp.word_vector.sentence_vectorizer(text, use_mean=True)[source]¶

Get sentence vector from text If a word is not in the vocabulary, KeyError will be raised. :param str text: text input :param boolean use_mean: if True use mean of all word vectors else use summation :return: sentence vector of given input text
